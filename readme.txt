Lists current WordPress plugins used inside of page/post content.

== Description ==

By placing <!--plugin list--> in your page or post by sources code or using toolbar icon 'Plugin List' in 
your Rich text editor(only if enabled) plugin replaces it with list of currently installed WordPress plugins.
You can use various configuration options for tweaking plugin list.

== Installation ==

Upload the plugin directory to your wp-content/plugins/ and activate it. After plugin is active
insert the following into any post or page: <!--plugin list--> and save it. All done.
